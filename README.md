# 🍕 FoodieExpress - Online Food Delivery App

Welcome to **FoodieExpress**, a lightweight, responsive web application designed for a seamless food ordering experience. This project was built to demonstrate modern front-end development skills, including dynamic filtering, real-time search, and a functional shopping cart.

## 🚀 Live Demo
You can view the live project here: [https://manojbsmanojbs390-png.github.io/my-food-delivery-app/]

---

## ✨ Key Features
* **Dynamic Search:** Find your favorite meals instantly using the real-time search bar.
* **Category Filtering:** Sort the menu by 'Fast Food' or 'Healthy' categories.
* **Live Shopping Cart:** Add items to your bag and watch the total bill update automatically.
* **Responsive Design:** Fully optimized for Desktop, Tablet, and Mobile screens.
* **Star Ratings:** Visual feedback system showing customer ratings and review counts.

---

## 🛠️ Technologies Used
* **HTML5:** For the structural skeleton of the application.
* **CSS3:** For professional styling, Flexbox layouts, and smooth transitions.
* **JavaScript (ES6):** For the application logic, DOM manipulation, and filtering systems.

---

## 📁 Project Structure
* `index.html` - The main entry point and structural code.
* `style.css` - Contains all custom designs and mobile-responsive rules.
* `script.js` - Handles the search, filter, and cart calculations.

---

## 📖 How to Use
1.  **Search** for a food item in the top search bar.
2.  Use the **Category Buttons** to filter between different food types.
3.  Click **Add to Cart** to see your total bill grow in real-time.
4.  Click **Place Order Now** to complete your simulated checkout!
